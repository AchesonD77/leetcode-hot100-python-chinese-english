# 正确刷题方法（最重要）

```text
1. 先自己想
2. 写暴力解
3. 再优化
4. 看题解
5. 第二天重写
```

---

# 最终目标

看到题后形成条件反射：

| 题目特征 | 第一反应 |
|---|---|
| target - x | hashmap |
| 左右夹逼 | 双指针 |
| 动态区间 | 滑动窗口 |
| 递归遍历 | DFS |
| 分层遍历 | BFS |
| 最优子结构 | DP |
| 局部最优 | 贪心 |
| 单调关系 | 二分 |
| 最近更大元素 | 单调栈 |

---
