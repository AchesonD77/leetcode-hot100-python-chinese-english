# English Version

## 1. Problem Core

Given an array:

```python
nums = [-1,0,1,2,-1,-4]
```

Find all combinations where:

```text
three numbers sum to 0
```

Requirements:

- No duplicate answers
- Return all valid combinations

---

## 2. Core Idea (Sorting + Two Pointers)

Steps:

### 1. Sort the Array First

```python
nums.sort()
```

After sorting:

```python
[-4,-1,-1,0,1,2]
```

---

### 2. Fix One Number

For example:

```python
nums[i]
```

Then use:

- one left pointer
- one right pointer

to find:

```text
nums[left] + nums[right] = -nums[i]
```

---

## 3. Why Sorting Is Necessary (Important)

After sorting, we can:

- use two pointers
- quickly decide which pointer to move
- remove duplicates efficiently

This is the core of the problem.

If any of the three positions are duplicated,
duplicate answers may appear.

Therefore:

```text
the fixed position and both pointers all need deduplication
```

---

## 4. Python Code (Must Memorize)

```python
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        nums.sort()

        result = []

        n = len(nums)

        for i in range(n):

            # remove duplicates
            if i > 0 and nums[i] == nums[i - 1]:
                continue

            left = i + 1
            right = n - 1

            while left < right:

                total = nums[i] + nums[left] + nums[right]

                if total < 0:
                    left += 1

                elif total > 0:
                    right -= 1

                else:
                    result.append([nums[i], nums[left], nums[right]])

                    # remove left duplicates
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1

                    # remove right duplicates
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1

                    left += 1
                    right -= 1

        return result
```

### 2. Left Pointer Deduplication


```python

```
