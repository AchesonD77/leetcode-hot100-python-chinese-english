# LeetCode 239. 滑动窗口最大值（Sliding Window Maximum）

## 题目描述

![alt text](<../../Snap_for_Questions/4_Substring_子串/截屏2026-06-06 13.48.17.png>)

```python

class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        # 创建双端列表 double que
        q = deque()
        # result is a array
        res = []

        # loop
        for i in range(len(nums)):
            # 删除往右边移动的时候，左边已经离开k滑动窗口的元素。q[0] 表示的是q列表中对应的第一位index对应的 nums 的下标值
            if q and q[0] <= i - k:
                q.popleft() # 默认删除左边元素的函数

            # 删除左边后，开始判断列表最右边的是否为比 更新的值 大，来确认最大值始终在最前面：
            # 维护单调递减函数
            while q and nums[q[-1]] < nums[i]:
                q.pop() # 删除q列表的最右边的元素
            
            q.append(i)
            # 已经开始形成k滑动列表：
            if i >= k - 1:
                res.append(nums[q[0]])
        return res

```
        
    


给定一个整数数组 `nums` 和一个大小为 `k` 的滑动窗口。

窗口从数组最左侧移动到最右侧，每次向右移动一格。

要求返回：

```text
每个窗口中的最大值
```

---

```text
 1  3 -1 -3  5 [3  6  7]   -> 7
```


# 七、最优解代码（LeetCode）

```python
from collections import deque

class Solution:
    def maxSlidingWindow(self, nums, k):
        q = deque()
        res = []

        for i in range(len(nums)):

            # 删除已经离开窗口的元素
            if q and q[0] <= i - k:
                q.popleft()

            # 维护单调递减队列
            while q and nums[q[-1]] < nums[i]:
                q.pop()

            q.append(i)

            # 窗口形成
            if i >= k - 1:
                res.append(nums[q[0]])

        return res
```

# 八、代码讲解（Code Explanation）

## 创建双端队列

```python
q = deque()
```

q = deque()
创建一个双端队列，用来存储窗口中的元素下标，支持 O(1) 时间复杂度的队首删除（popleft）和队尾删除（pop），是实现单调队列和滑动窗口问题的核心数据结构。

存储：

```text
元素下标，队列里面存的不是值，是nums的下标index
nums = [1,3,-1,-3,5]
q = [1,2,3]
那么：
q[0] = 1
q[1] = 2
q[2] = 3
这些都是：nums的下标
对应值：
nums[1] = 3
nums[2] = -1
nums[3] = -3

q = [1,2,3]
实际对应
[3,-1,-3]
```

---

## 删除过期元素

```python
if q and q[0] <= i-k:
    q.popleft()
```

作用：

```text
删除已经离开窗口的最大值
```

---

例如：

```python
k = 3
i = 5
窗口左边界：i-k+1
```

当前窗口：

```python
[3,4,5]
```

如果：

```python
q[0] = 2 表示 nums[2]
```

说明：

```text
下标2已经离开窗口
```

删除。

q[0] ： 当前窗口最大值对应元素的下标
nums[q[0]] ： 当前窗口最大值

队列存下标，用下标判断是否出窗口；用 nums[q[0]] 获取窗口最大值。

```text
为什么要先判断 q？

队列里面存的不是值。
存的是：下标(index)

q[0]
表示访问 存储的nums列表的index下标的 队列中 的第一个元素。
如果队列为空：会直接报错
所以必须先判断：if q: 队列非空?
等价于 if len(q) > 0:

```
```text
if q and q[0] <= i - k:
这是 Python 的短路运算（Short Circuit）。
从左往右 两部分，左边第一个为false，不会执行后边部分。

```



---

## 维护单调递减，这一行代码是 单调队列算法的灵魂

```python
while q and nums[q[-1]] < nums[i]:
    q.pop()
```

作用：

```text
删除所有比当前元素小的数

把所有比当前元素小的元素全部踢掉，因为它们以后永远不可能成为窗口最大值。
```

保证：

```text
队首最大
队尾最小
```

```text
这行代码在比较什么？
nums[q[-1]] 队尾元素的值, q[-1] 表示存储的nums下标的最后一位。
nums[i] 当前新来的元素

表示： while 队尾值 < 当前值:

当前元素更大时，把队尾所有更小元素全部删除，因为它们既没有当前元素大，又会比当前元素更早离开窗口，因此永远不可能成为未来窗口最大值。这样就能保证队列始终单调递减，队首永远是窗口最大值。

q.pop() 是 从队列（deque）的右边删除一个元素。
q.pop() 删除最右边元素
q.popleft() 删除最左边元素
```

## 记录答案

```python
if i >= k - 1:
    res.append(nums[q[0]])
```
```text
是整个算法最后一步：
当窗口形成后，把当前窗口的最大值加入答案。

第一步：为什么要判断 i >= k - 1？
终于有 3 个元素了。第一个完整窗口出现。 此时才能开始记录答案。意思就是：当前已经形成了一个完整窗口。

第二步：为什么是 nums[q[0]]？
nums[q[0]]才是窗口最大值。

为什么队首一定是最大值？
因为前面维护了
其实就是：把当前窗口最大值加入答案

每当形成一个完整窗口，就把单调队列队首对应的最大值记录下来。
```

窗口形成后：

```python
q[0]
```

永远指向：

```text
当前窗口最大值
```


因此：

```text

```

总入队次数：

```text

```

```text
n
```

最终：

```text
O(n)
```
