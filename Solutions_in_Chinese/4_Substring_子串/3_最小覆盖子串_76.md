# LeetCode 76. 最小覆盖子串（Minimum Window Substring）

---

# 一、题目描述
![alt text](<../../Snap_for_Questions/4_Substring_子串/截屏2026-06-07 16.42.09.png>)

给定两个字符串：

```python
s
t
```

要求在字符串 `s` 中找到一个最短连续子串，使得：

```text
该子串包含 t 中所有字符

并且字符出现次数也满足要求
```

如果不存在这样的子串：

```python
return ""
```

---

## 示例 1

```python
s = "ADOBECODEBANC"
t = "ABC"
```

输出：

```python
"BANC"
```

解释：

```text
BANC 包含：

A
B
C

并且长度最短
```

---

## 示例 2

```python
s = "a"
t = "a"
```

输出：

```python
"a"
```

---

## 示例 3

```python
s = "a"
t = "aa"
```

输出：

```python
""
```

# 三、暴力解法

枚举所有子串：

```python
for i in range(n):
    for j in range(i,n):
        判断 s[i:j]
```

---

时间复杂度：

```text
O(n³)

原因：
枚举：O(n²)

判断：O(n)

总计：O(n³)
```

无法通过。

# 四、本题核心思想

使用：

```text
滑动窗口 + 哈希表
```

窗口：

```text
[left ... right]
```

不断向右扩张：

```text
right →
```

直到：

```text
窗口已经覆盖 t
```

然后：

```text
left →
```

不断缩小窗口。

因为：

```text
我们要找最短覆盖子串
```

# 五、核心数据结构

本题维护两个哈希表：

```python
need
window
```

---

## need

记录：

```text
t 中每个字符需要出现多少次
```

例如：

```python
t = "AABC"
```

得到：

```python
{
    'A': 2,
    'B': 1,
    'C': 1
}
```

---

## window

记录：

```text
当前窗口中的字符数量
```

例如：

```python
{
    'A': 2,
    'B': 1,
    'C': 1
}
```

# 六、valid 的意义

这是本题最关键的变量。

---

定义：

```python
valid
```

表示：

```text
已经满足要求的字符种类数
```

---

例如：

```python
need = {
    'A':1,
    'B':1,
    'C':1
}
```

当前窗口：

```python
window = {
    'A':1,
    'B':1,
    'C':1
}
```

则：

```python
valid = 3
```

---

因为：

```text
A 满足

B 满足

C 满足
```

---

此时：

```python
valid == len(need)
```

说明：

```text
窗口已经覆盖 t
```

# 七、为什么不用直接比较两个哈希表？

很多同学会写：

```python
if window == need:
```

问题：

```text
效率太低
```

每次比较：

```text
O(k)
```

---

valid 的优势：

```python
if valid == len(need):
```

仅：

```text
O(1)
```

# 八、算法流程

## Step1

统计目标字符

```python
need = Counter(t)
```

得到：

```python
{
    'A':1,
    'B':1,
    'C':1
}
```

---

## Step2

右指针扩张

```python
right += 1
```

不断加入字符。

---

## Step3

更新窗口

```python
window[c] += 1
```

---

## Step4

更新 valid

```python
if window[c] == need[c]:
    valid += 1
```

说明：

```text
字符 c 达标
```

---

## Step5

判断是否覆盖

```python
valid == len(need)
```

若成立：

```text
窗口已经覆盖 t
```

---

## Step6

开始收缩窗口

```python
left += 1
```

寻找更短答案。

# 十、LeetCode 核心代码模式

这是面试最常见写法。

```python
from collections import Counter, defaultdict

class Solution:
    def minWindow(self, s: str, t: str) -> str:

        need = Counter(t)
        window = defaultdict(int)

        left = 0
        right = 0

        valid = 0

        start = 0
        min_len = float("inf")

        while right < len(s):

            c = s[right]
            right += 1

            if c in need:

                window[c] += 1

                if window[c] == need[c]:
                    valid += 1

            while valid == len(need):

                if right - left < min_len:
                    start = left
                    min_len = right - left

                d = s[left]
                left += 1

                if d in need:

                    if window[d] == need[d]:
                        valid -= 1

                    window[d] -= 1

        return "" if min_len == float("inf") else s[start:start+min_len]
```
