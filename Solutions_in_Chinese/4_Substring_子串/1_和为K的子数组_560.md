# LeetCode 560. 和为 K 的子数组（Subarray Sum Equals K）

## 题目描述

![alt text](<../../Snap_for_Questions/4_Substring_子串/截屏2026-06-05 15.17.48.png>)

```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        # 使用哈希表-初始默认值
        check = defaultdict(int)
        # 一开始：空前缀为0的时候，出现了一次记录为一次。
        check[0] = 1 
        # 变量存储值，前缀和，答案和
        pre = 0
        count = 0

        # 遍历循环 数组列表 所有的值
        for num in nums:
            # 前缀和
            pre += num
            # 寻找 当前前缀和 - k = 之前的可以构成结果和为整数k的 前缀的 位置
            count += check[pre - k]
            # 出现的 前缀和 和 次数 记录到哈希表中
            check[pre] += 1
        # 返回 循环遍历 全部数组后的 能够组成和为k 的次数
        return count


```



给定一个整数数组 `nums` 和一个整数 `k`，请你统计并返回该数组中和为 `k` 的连续子数组的个数。

**子数组** 是数组中元素的连续非空序列。

# 一、暴力解法

思路：

枚举所有可能的连续子数组。

```python
count = 0

for i in range(len(nums)):
    s = 0

    for j in range(i, len(nums)):
        s += nums[j]

        if s == k:
            count += 1

return count
```

---

## 时间复杂度

```text
O(n²)
```

原因：

```text
每个起点
    枚举所有终点
```

---

## 空间复杂度

```text
O(1)
```

# 三、核心思想：前缀和（Prefix Sum）

## 什么是前缀和？

定义：

```python
pre[i]
```

表示：

```text
nums[0] ~ nums[i]
的元素总和
```

---

例如：

```python
nums = [1,2,3,4]
```

前缀和数组：

```text
pre[0] = 1

pre[1] = 1 + 2 = 3

pre[2] = 1 + 2 + 3 = 6

pre[3] = 1 + 2 + 3 + 4 = 10
```

结果：

```python
[1,3,6,10]
```

**为什么有 mp[0] = 1**

表示在数组开始之前，
存在一个前缀和为0的位置。

这样才能正确统计
从下标0开始的子数组。

否则会漏掉
前缀和本身就等于k的情况。



前缀和问题中：

mp[0] = 1

不是特殊技巧。

而是在表示：

"空前缀（Empty Prefix）出现过一次。"


---

# 四、关键数学公式

假设：

```text
pre[j]
```

表示：

```text
nums[0] ~ nums[j]
的和
```

那么：

```text
nums[i] ~ nums[j]
```

的和为：

```text
pre[j] - pre[i-1]
```

即：

```math
pre[j] - pre[i-1] = k
```

整理得：

```math
pre[i-1] = pre[j] - k
```

---

## 核心理解

如果当前前缀和：

```python
pre = 10
```

目标：

```python
k = 4
```

那么只要以前出现过：

```python
10 - 4 = 6
```

**说明：** 核心意思

```text
从那个前缀和为6的位置之后
到当前位置之间

这一段的和一定等于4
```

# 八、最优解代码（面试必背）

```python
from collections import defaultdict

class Solution:
    def subarraySum(self, nums, k):
        mp = defaultdict(int)

        mp[0] = 1

        pre = 0
        count = 0

        for num in nums:

            pre += num

            count += mp[pre - k]

            mp[pre] += 1

        return count
```

---

# 核心算法：为什么 `count += mp[pre - k]` 能成立？

这是整道题最核心的一行代码：

```python
count += mp[pre - k]
```

---

## 核心含义

```text
看看以前有多少个前缀和
等于 pre-k。

这些位置都可以作为当前区间的起点。

有多少个，就直接给答案加多少个。
```

---

## 例子1

假设：

```python
pre = 3
k = 2
```

那么：

```python
pre - k = 1
```

此时我们要问：

```text
以前有没有出现过前缀和 = 1？
```

如果：

```python
mp[1] = 1
```

表示：

```text
以前有一个位置
前缀和等于 1。
```

由于：

```text
3 - 1 = 2
```

所以：

```text
找到了一个区间和等于 k 的子数组。
```

因此：

```python
count += 1
```

---

## 例子2

如果：

```python
mp[1] = 3
```

表示：

```text
以前有三个位置
前缀和都等于 1。
```

由于：

```text
3 - 1 = 2
```

对于这三个位置都成立。

因此：

```text
可以形成三个不同的合法区间。
```

所以：

```python
count += 3
```

---

# 最重要的公式

区间和公式：

```text
区间和
=
当前前缀和
-
以前前缀和
```

即：

```text
k = pre - old_pre
```

移项得到：

```text
old_pre = pre - k
```

---

# 代码到底在问什么？

```python
count += mp[pre - k]
```

本质是在问：

```text
以前有多少个前缀和
等于 pre-k？
```

因为：

```text
每出现一个这样的前缀和

都能组成一个：

和为 k 的连续子数组
```

所以：

```text
出现几次

答案就增加几次
```

直接加入答案即可。

---
