# 中文版本

## 一、题目核心

给定数组：

```python
height = [0,1,0,2,1,0,1,3,2,1,2,1]
```

每个数字表示柱子高度。

求：

```text
下雨后最多能接多少雨水
```

---

## 二、雨水计算核心（最重要）

当前位置能接多少水：

```text
取决于：
左边最高柱子
右边最高柱子
```

当前位置雨水高度：

```text
min(左边最大高度, 右边最大高度)
```

减去：

```text
当前柱子高度
```

所以：

```text
当前位置雨水 =
min(left_max, right_max) - height[i]
```

---

## 三、核心思想（双指针）

使用两个指针：

- `left`
- `right`

同时维护：

- `left_max`
- `right_max`

每次：

```text
谁小移动谁
```

因为：

```text
短板决定当前能接多少水
```

---

## 四、为什么移动较小的一边（重点）

假设：

```text
left_max < right_max
```

说明：

```text
左边已经确定是短板
```

当前位置能接多少水：

```text
只由 left_max 决定
```

所以：

```text
可以安全计算左边雨水
```

然后移动：

```python
left += 1
```

---

## 五、Python 代码（LeetCode版本）

```python
class Solution:
    def trap(self, height: list[int]) -> int:

        left = 0
        right = len(height) - 1

        left_max = 0
        right_max = 0

        result = 0

        while left < right:

            left_max = max(left_max, height[left])
            right_max = max(right_max, height[right])

            if left_max < right_max:

                result += left_max - height[left]

                left += 1

            else:

                result += right_max - height[right]

                right -= 1

        return result
```

---

### 4. while 循环

```python

```

双指针经典写法。

**for 适用于“知道循环多少次”，while 适用于“知道什么时候结束”；双指针题通常只知道“指针相遇时结束”，所以用 while。**
**双指针：
指针移动方向动态决定
循环次数未知
结束条件是指针相遇**
