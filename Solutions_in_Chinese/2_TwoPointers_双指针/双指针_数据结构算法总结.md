# LeetCode Hot100 专题总结 —— 双指针（Two Pointers）

---

# 一、什么是双指针（Two Pointers）

双指针：

```text
使用两个指针协同工作
通过移动指针减少暴力搜索
从而降低时间复杂度
```

本质：

```text
空间换时间
暴力优化
```

很多：

```text
O(n²)
O(n³)
```

的问题，

都可以优化到：

```text
O(n)
O(n²)
```

---

# 二、为什么要学双指针

面试出现频率极高。

# 三、双指针的四大类型

## 类型1：快慢指针

特点：

```text
一个快
一个慢
```

用途：

```text
原地修改数组
删除元素
覆盖元素
```

典型题：

### 283. 移动零

```python
slow = 0

for fast in range(len(nums)):
    if nums[fast] != 0:
        nums[slow], nums[fast] = nums[fast], nums[slow]
        slow += 1
```

核心：

```text
fast 找元素
slow 放元素
```

---

### 模板

```python
slow = 0

for fast in range(len(nums)):

    if 满足条件:

        nums[slow] = nums[fast]

        slow += 1
```

---

## 类型2：左右夹逼的双指针

特点：

```text
left ←
      →
       right
```

两端向中间靠拢。

用途：

```text
有序数组
区间最优解
```

典型题：

### 11. 盛最多水的容器

```python
left = 0
right = n - 1

while left < right:
```

核心：

```text
移动短板
```

---

### 模板

```python
left = 0
right = n - 1

while left < right:

    if 条件:
        left += 1
    else:
        right -= 1
```

---

## 类型3：固定一个数 + 双指针

特点：

```text
一个数固定
另外两个数双指针寻找
```

用途：

```text
3Sum
4Sum
KSum
```

典型题：

### 15. 三数之和

```python
for i in range(n):

    left = i + 1
    right = n - 1
```

核心：

```text
排序 + 双指针
```

---

### 模板

```python
nums.sort()

for i in range(n):

    left = i + 1
    right = n - 1

    while left < right:

        total = nums[i] + nums[left] + nums[right]

        if total < target:
            left += 1

        elif total > target:
            right -= 1

        else:
            # 找到答案
```

---

## 类型4：左右边界统计

特点：

```text
同时维护左右信息
```

用途：

```text
接雨水
区间统计
前后缀问题
```

典型题：

### 42. 接雨水

维护：

```python
left_max
right_max
```

核心：

```text
谁低移动谁
```

---

### 模板

```python
left = 0
right = n - 1

left_max = 0
right_max = 0

while left < right:

    left_max = max(left_max, nums[left])
    right_max = max(right_max, nums[right])

    if left_max < right_max:

        left += 1

    else:

        right -= 1
```

# 四、双指针的核心思想

---

## 思想1：快慢指针

```text
fast 找元素
slow 放元素
```

典型：

```text
移动零
删除元素
删除重复项
```

---

## 思想2：左右夹逼

```text
left 从左
right 从右
```

不断缩小搜索空间。

典型：

```text
盛最多水
两数之和II
回文串
```

---

## 思想3：移动短板

典型：

### 11 盛最多水

```text
面积 = 宽 × 高
```

高度：

```text
由短板决定
```

所以：

```text
移动短板
```

---

### 42 接雨水

```text
雨水高度
由较低边界决定
```

所以：

```text
移动较低边界
```

---

## 思想4：排序后双指针

典型：

### 15 三数之和

排序后：

```text
和太小
left++
```

```text
和太大
right--
```

# 六、面试ACM模板（万能双指针）

```python
nums = list(map(int, input().split()))

left = 0
right = len(nums) - 1

while left < right:

    if 条件满足:

        left += 1

    else:

        right -= 1
```
