# 一、题目描述

给你一个链表头节点 `head`，以及一个整数 `k`。

要求：

> 每 `k` 个节点作为一组进行翻转，最后返回翻转后的链表。

注意：

- 只能修改节点之间的连接关系（next）
- 不能修改节点的值（val）
- 如果最后剩余节点数量不足 `k`，保持原来的顺序

---

## 示例1

输入：

```
head = [1,2,3,4,5]
k = 2
```

原链表：

```
1 → 2 → 3 → 4 → 5
```

每2个一组：

```
(1 2)

(3 4)

(5)
```

翻转：

```
2 → 1

4 → 3

5
```

最终：

```
2 → 1 → 4 → 3 → 5
```

---

## 示例2

输入：

```
head = [1,2,3,4,5]
k = 3
```

分组：

```
(1 2 3)

(4 5)
```

翻转第一组：

```
3 → 2 → 1
```

第二组不足3个：

保持：

```
4 → 5
```

最终：

```
3 → 2 → 1 → 4 → 5
```

# 八、LeetCode 核心代码模板（★★★★★）

```python
class Solution:
    def reverseKGroup(self, head: ListNode, k: int) -> ListNode:

        dummy = ListNode(0)
        dummy.next = head

        pre = dummy

        while True:

            # 找当前组尾节点
            tail = pre
            for _ in range(k):
                tail = tail.next
                if not tail:
                    return dummy.next

            # 保存下一组
            next_group = tail.next

            # 翻转当前组（reverse函数将在后面详细讲解）
            newHead, newTail = self.reverse(pre.next, tail)

            # 重新连接
            pre.next = newHead
            newTail.next = next_group

            # pre移动到下一组
            pre = newTail
```

# 十一、reverse() 子函数完整代码（★★★★★）

```python
def reverse(self, head: ListNode, tail: ListNode):

    prev = tail.next
    cur = head

    while prev != tail:

        nxt = cur.next
        cur.next = prev
        prev = cur
        cur = nxt

    return tail, head
```

这是整个 LeetCode 25 最核心的一段代码。

很多人第一次看到：

```python

```

都会懵。

# 十七、完整 LeetCode 代码（★★★★★）

```python
class Solution:

    def reverse(self, head, tail):

        prev = tail.next
        cur = head

        while prev != tail:

            nxt = cur.next
            cur.next = prev
            prev = cur
            cur = nxt

        return tail, head

    def reverseKGroup(self, head, k):

        dummy = ListNode(0)
        dummy.next = head

        pre = dummy

        while True:

            tail = pre

            for _ in range(k):

                tail = tail.next

                if not tail:
                    return dummy.next

            nxt = tail.next

            newHead, newTail = self.reverse(pre.next, tail)

            pre.next = newHead

            newTail.next = nxt

            pre = newTail
```

# 十九、涉及的重要 Function / Python 语法讲解

## 1. `reverse(head, tail)`

作用：只反转从 `head` 到 `tail` 的这一段链表，并返回翻转后的新头节点和新尾节点。

---

## 2. `tail.next`

作用：表示当前这一组后面的第一个节点。例如：

```
1 → 2 → 3 → 4

tail = 3

tail.next = 4
```

---

## 3. `return tail, head`

Python 支持一次返回多个值。

例如：

```python
return tail, head
```

接收：

```python
newHead, newTail = reverse(...)
```

等价于：

```python
newHead = tail
newTail = head
```

---

## 4. `while True`

作用：无限循环，直到：

```python
return dummy.next
```

主动结束。

---

# 二十、为什么连接顺序不能乱？（★★★★★）

正确顺序：

```python
next_group = tail.next

newHead, newTail = reverse(...)

pre.next = newHead

newTail.next = next_group
```

如果：

先：

```python
tail.next = None
```

或者：

先：

```python
pre = newTail
```

都可能导致：

- 链表断开
- 丢失下一组
- 形成环

因此：

连接顺序必须严格按照模板。
