# LeetCode 24. 两两交换链表中的节点（Swap Nodes in Pairs）

---

# 一、题目描述

![alt text](<../../Snap_for_Questions/7_Linked_Lists_链表/截屏2026-06-30 12.48.00.png>)

快速答案（本题解析有两种方法，我们选择 递归法）
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        # 本题我们选择使用递归方法，掌握这个思想
        # 首先要确保要交换的两个节点非空
        # 判断语句可以理解为：if head 意味着如果 head非None。if not head理解为 如果 head 是 None。
        if not head or not head.next:
            return head
        # 先存储1和2的位置节点：
        first = head
        second = head.next
        # 开始递归 - 第三位数递归（让第三位和第四位交换），然后第一位接到第三位，成为第二位
        first.next = self.swapPairs(second.next)
        second.next = head
        
        return second

```

给你一个单链表的头节点 `head`。

要求：

> 每两个相邻节点交换一次，并返回交换后的链表。

**注意：**

- **不能修改节点的值（val）**
- **只能修改节点之间的指针（next）**

例如：

```
原链表：

1 → 2 → 3 → 4
```

交换后：

```
2 → 1 → 4 → 3
```

---

## 示例1

输入：

```
head = [1,2,3,4]
```

输出：

```
[2,1,4,3]
```

---

## 示例2

输入：

```
head = []
```

输出：

```
[]
```

---

## 示例3

输入：

```
head = [1]
```

输出：

```
[1]
```

因为只有一个节点，无法交换。

# 二、本题本质

很多同学第一反应：

```
交换两个节点的值（val）
```

例如：

```
1 ↔ 2
```

交换：

```
val
```

但是题目明确要求：

> **不能修改节点值，只能修改节点之间的连接关系（next）。**

因此：

真正需要交换的是：

```
1 → 2
```

变成：

```
2 → 1
```

这就是链表指针操作。

---

# 三、本题核心思想（★★★★★）

采用：

> **Dummy（虚拟头节点） + 指针操作**

整体流程：

```
建立 Dummy
↓
找到需要交换的一对节点
↓
调整三个 next
↓
移动到下一组
↓
重复直到结束
```

这是链表节点交换最经典的模板。

# 四、为什么需要 Dummy？

例如：

```
1 → 2 → 3 → 4
```

交换第一组后：

```
2 → 1 → 3 → 4
```

新的头节点已经变成：

```
2
```

如果没有 Dummy，需要专门修改：

```

```

比较麻烦。

加入 Dummy：

```
dummy
↓
0 → 1 → 2 → 3 → 4
```

以后所有交换都统一处理。

最后返回：

```python
dummy.next
```

即可。

# 六、LeetCode 核心代码模式（★★★★★）

```python
class Solution:
    def swapPairs(self, head: ListNode) -> ListNode:

        # 创建虚拟头节点
        dummy = ListNode(0)
        dummy.next = head

        # 当前节点（每组前一个节点）
        cur = dummy

        # 至少还剩两个节点才能交换
        while cur.next and cur.next.next:

            # 当前两个待交换节点
            first = cur.next
            second = cur.next.next

            # 第一步：first 指向下一组
            first.next = second.next

            # 第二步：second 指向 first
            second.next = first

            # 第三步：cur 指向 second
            cur.next = second

            # cur 移动到下一组前驱
            cur = first

        return dummy.next
```

# 八、为什么交换顺序不能乱？

```python

```

```python

```

再修改其它指针。

容易：

```

```

或者：

形成：

```

```

因此：

一定按照模板顺序修改。

class Solution:
    def swapPairs(self, head: ListNode) -> ListNode:

        if not head or not head.next:
