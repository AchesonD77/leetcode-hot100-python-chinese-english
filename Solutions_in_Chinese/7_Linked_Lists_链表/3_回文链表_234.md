# LeetCode 234. 回文链表（Palindrome Linked List）

## 一、题目描述

![alt text](<../../Snap_for_Questions/7_Linked_Lists_链表/截屏2026-06-23 12.00.17.png>)

给你一个单链表的头节点 `head`。

请判断该链表是否为回文链表。

如果是：

```python
return True
```

否则：

```python
return False
```

---

## 什么是回文？

回文（Palindrome）指：

> 正着读和反着读完全相同。

例如：

```text
121
1221
12321
```

都是回文。

---

### 示例1

链表：

```text
1 → 2 → 2 → 1
```

反过来：

```text
1 → 2 → 2 → 1
```

相同。

返回：

```python
True
```

---

### 示例2

链表：

```text
1 → 2
```

反过来：

```text
2 → 1
```

不同。

返回：

```python
False
```

---

# 二、本题核心思想

数组判断回文非常简单：

```python
nums == nums[::-1]
```

但是链表没有下标。

因此必须：

```text
找到中点
↓
反转后半部分
↓
前后比较
```

这是链表回文题最经典的解法。

---

# 三、最优解思路（★★★★★）

利用：

## 快慢指针 + 链表反转

步骤：

```text
① 找中点

② 反转后半链表

③ 前后同时比较

④ 全部相同返回 True

⑤ 出现不同返回 False
```

时间复杂度：

```text
O(n)
```

空间复杂度：

```text
O(1)
```

满足题目进阶要求。

# 五、LeetCode 核心代码模式（★★★★★）

```python
class Solution:
    def isPalindrome(self, head: ListNode) -> bool:

        # 1. 找中点
        slow = head
        fast = head

        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next


        # 2. 反转后半部分
        prev = None
        curr = slow

        while curr:
            nxt = curr.next
            curr.next = prev
            prev = curr
            curr = nxt


        # 3. 前后比较
        left = head
        right = prev

        while right:

            if left.val != right.val:
                return False

            left = left.next
            right = right.next


        return True
```

# 六、逐句代码讲解

---

## 第一部分：寻找中点

### 初始化

```python
slow = head
fast = head
```

含义：

```text
slow 一次走一步

fast 一次走两步
```

---

### 快慢指针循环

```python
while fast and fast.next:
```

作用：

保证：

```python
fast.next.next
```

不会越界。

---

### slow走一步

```python
slow = slow.next
```

---

### fast走两步

```python
fast = fast.next.next
```

---

最终：

```text
slow
```

来到链表中间位置。

### why?
```text
为什么一定是中点？

关键数学关系：假设：

slow 走了 k 步

fast 走了 2k 步

当 fast 到达尾部时：

fast ≈ n
其中：
n = 链表长度

因为：fast = 2 × slow

所以：slow = n / 2

这就是中点。l
```

# 第二部分：反转后半链表

---

### 初始化

```python
prev = None
curr = slow
```

表示：

```text
从中点开始反转
```

---

### 反转循环

```python
while curr:
```

只要还有节点。

继续反转。

---

### 保存后继节点

```python
nxt = curr.next
```

作用：

防止链表断掉。

---

### 反转指针

```python
curr.next = prev
```

例如：

原来：

```text
2 → 1
```

变成：

```text
2 ← 1
```

---

### 移动 prev

```python
prev = curr
```

记录当前反转后的头。

---

### 移动 curr

```python
curr = nxt
```

继续处理下一个节点。

---

循环结束：

```text
prev
```

就是反转后链表头。

### 理解
```text
很多初学者最大的误区是：

认为 next_node = curr.next 是复制了一个节点。

实际上：

链表变量存储的是节点引用（地址）

你可以理解为：

next_node
prev
curr
head

这些变量里面装的都是：

节点地址

而不是节点值。

含义：

把当前节点的箭头
改成指向前一个节点

```

发现不同：

### 同时向前移动

```python

```

意思：
沿着链表向后移动一个节点

全部存的是：
节点引用（地址）

---

```python

```





比较：

```text

2 == 2
3 == 3
```
