# LeetCode 2. 两数相加（Add Two Numbers）

# 一、题目描述
![alt text](<../../Snap_for_Questions/7_Linked_Lists_链表/截屏2026-06-27 18.08.33.png>)

直接答案：
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        # 使用虚拟头结点，cur指针，进位变量来进行判断、存储、执行算法
        # 虚拟头结点
        dummy = ListNode()
        # 当前尾节点
        cur = dummy
        # 进位变量
        carry = 0

        # 开始循环
        while l1 or l2 or carry:
            # 逆序，从个位开始：
            x = l1.val if l1 else 0
            y = l2.val if l2 else 0

            # 计算总和
            total = x + y + carry
            # 更新进位：// 取十位法
            carry = total // 10
            # 保存当前节点
            cur.next = ListNode(total % 10)
            # cur后移指向下一位
            cur = cur.next
            # l1、l2后移
            if l1:
                l1 = l1.next
            if l2:
                l2 = l2.next
        return dummy.next
```

给你两个**非空链表**：

```text
l1
l2
```

它们分别表示两个非负整数。

注意：

- 每个节点只保存一位数字（0~9）
- 数字按照**逆序存储**

例如：

```
342
```

存储成：

```
2 → 4 → 3
```

因为：

```
个位 → 十位 → 百位
```

现在要求：

```
返回两个数字相加后的链表。
```

# 六、LeetCode 核心代码模式（★★★★★）

```python
class Solution:
    def addTwoNumbers(
        self,
        l1: ListNode,
        l2: ListNode
    ) -> ListNode:

        # 虚拟头节点
        dummy = ListNode()

        # 当前尾节点
        cur = dummy

        # 进位
        carry = 0


        while l1 or l2 or carry:

            # 当前两位数字
            x = l1.val if l1 else 0
            y = l2.val if l2 else 0

            # 当前位总和
            total = x + y + carry

            # 更新进位
            carry = total // 10

            # 当前节点保存个位
            cur.next = ListNode(total % 10)

            # cur 后移
            cur = cur.next

            # l1 后移
            if l1:
                l1 = l1.next

            # l2 后移
            if l2:
                l2 = l2.next


        return dummy.next
```

# 易错总结

cur.next = ListNode(total % 10)
为什么不能写成：
cur.next = total % 10
- 因为链表中的 next 存放的是"下一个节点（Node）"，不是一个数字（int）。

先理解 ListNode 是什么
```
ListNode
│
├── val   （保存值）
└── next  （保存下一个节点地址）

例如：node = ListNode(5)
实际上创建的是：
┌───────────────┐
│ val = 5       │
│ next = None   │
└───────────────┘
```
- 所以：ListNode(5)不是数字5。而是：一个节点对象

链表到底长什么样？
- 2 3 4
```text
┌───────────────┐
│ val = 2       │
│ next ─────────────┐
└───────────────┘   │
                    ▼
            ┌───────────────┐
            │ val = 4       │
            │ next ─────────────┐
            └───────────────┘   │
                                ▼
                        ┌───────────────┐
                        │ val = 3       │
                        │ next = None   │
                        └───────────────┘
```
- 注意：next 指向的是另一个节点。

```text
cur.next = ListNode(7)
变成：
dummy
  │
  ▼
┌───────────────┐
│ val = 7       │
│ next = None   │
└───────────────┘

现在：cur = cur.next

就是：cur = 节点7
```

- 链表有两个层次：
  - val 保存的是值（int），例如 7、0、8。
  - next 保存的是节点（ListNode），因此 cur.next 必须连接一个新的节点对象

```text
为什么可以写
x = l1.val if l1 else 0

而不能写
l1 = l1.val
```
- 首先理解 l1 是什么？l1不是数字2。而是：一个 ListNode 节点
- l1.val 表示：取节点里面保存的数字
- 所以：x = l1.val 意思就是：把数字2拿出来保存到 x
- 所以：l1 必须始终保存"节点"，不能变成数字。

```text
2. if l1: 表示什么？
```
等价于：
if l1 != None:

或者：
if l1 is not None:
- 意思：当前还有节点吗？

```text
3. 为什么最后
l1 = l1.next 正确？

因为：l1 永远应该指向：当前没有处理的那个节点.l1 = l1.next 的作用就是让 l1 沿着链表向后移动一个节点。
```

| 表达式           | 得到什么       | 类型         |
| ------------- | ---------- | ---------- |
| `l1`          | 当前节点       | `ListNode` |
| `l1.val`      | 当前节点保存的数字  | `int`      |
| `l1.next`     | 下一个节点      | `ListNode` |
| `l1.next.val` | 下一个节点保存的数字 | `int`      |

- l1 保存的是"节点（ListNode）"，l1.val 取得的是节点里的"值（int）"，l1.next 取得的是"下一个节点（ListNode）"。因此，读取数字时用 l1.val，遍历链表时必须用 l1 = l1.next，绝不能写成 l1 = l1.val，否则 l1 会从节点变成整数，后续再访问 .next 就会报错。
