# 一、题目描述

![alt text](<../../Snap_for_Questions/5_Normay_Array_普通数组/截屏2026-06-11 10.49.53.png>)


```python
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        
        # sort intervals
        intervals.sort(key = lambda x: x[0])
        # create a list variable to store the res and put first 0 position value inside
        res = [intervals[0]]

        # start to loop, from the 2nd-1 position
        for start, end in intervals[1:]:
            # we get the last one to store, -1 means the last one in the list, 1 means the second in [0,1]
            last_end = res[-1][1]
            # check
            if start <= last_end:
                # change the last end in the res list
                res[-1][1] = max(last_end, end)

            else:
                # bigger, so we just append it into the list
                res.append([start, end])

        return res

```

给定一个区间集合：

```python
intervals[i] = [start_i, end_i]
```

请你合并所有重叠的区间，并返回一个不重叠的区间数组。

要求：

```text
返回的区间需要完全覆盖原来的所有区间。
```

# 二、题目本质

题目本质：

```text
把所有有交集的区间合并成一个更大的区间。
```

例如：

```python
[1,3]
[2,6]
```

因为：

```text
2 <= 3
```

说明发生重叠。

可以合并：

```python
[1,6]
```

---

再看：

```python
[1,6]
[8,10]
```

因为：

```text
8 > 6
```

没有重叠。

所以：

```python
分别保留
```

---

# 三、核心思想

这题最关键的一步：

```text
先排序
```

---

例如：

```python
[[8,10],[1,3],[2,6]]
```

排序后：

```python
[[1,3],[2,6],[8,10]]
```

这样：

```text
只需要比较当前区间
和结果数组最后一个区间
```

即可。

# 五、判断区间是否重叠

假设：

```python
res[-1]
=

```


## 情况1：发生重叠

```text

```

例如：

```python
[1,4]
[3,6]
```

因为：

```text
3 <= 4
```

发生重叠。

---

此时更新终点：

```python
last_end = max(last_end, end)
```

得到：

```python
[1,6]
```

---

## 情况2：没有重叠

```text
start > last_end
```

例如：

```python
[1,4]
[8,10]
```

因为：

```text
8 > 4
```

没有交集。

直接加入结果：

```python
res.append([8,10])
```

## ACM代码

输入格式：

```text
4
1 3
2 6
8 10
15 18
```

表示：

```python
[
 [1,3],
 [2,6],
 [8,10],
 [15,18]
]
```

---

代码：

```python
def solve():

    n = int(input())

    intervals = []

    for _ in range(n):
        start, end = map(int, input().split())
        intervals.append([start, end])

    intervals.sort(key=lambda x: x[0])

    res = [intervals[0]]

    for start, end in intervals[1:]:

        if start <= res[-1][1]:

            res[-1][1] = max(
                res[-1][1],
                end
            )

        else:

            res.append([start, end])

    for start, end in res:
        print(start, end)


solve()
```
