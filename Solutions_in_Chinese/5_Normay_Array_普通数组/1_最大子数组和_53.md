# 三、核心思想（Kadane Algorithm）

这道题是动态规划经典题。

核心问题：

---

假设当前来到：

```python
nums[i]
```

那么：

```python
以 nums[i] 结尾的最大连续子数组和是多少？
```

---

例如：

```python
nums = [4,-1,2,1]
```

来到：

```python
2
```

此时有两种选择：

---

## 方案1：接上前面的连续子数组

```python
4 + (-1) + 2
```

得到：

```python
5
```

---

## 方案2：重新开始

只取自己：

```python
2
```

---

显然：

```python
max(5, 2)
```

取更大的。

---

## Kadane 最精华的一句话

对于位置 `i`：

```python
dp[i] = max(nums[i], dp[i - 1] + nums[i])
```

这并不是因为只存在两种连续子数组。

而是因为：

> 所有以 `i-1` 结尾的连续子数组，已经被 `dp[i-1]` 压缩成了一个「最优代表」。

---

因此，你不需要再枚举所有可能的前缀：

```text
[-1, 2]
[4, -1, 2]
[10, -3, -1, 2]
...
```

因为这些情况已经包含在：

```python
dp[i - 1]
```

之中了。


于是得到状态转移：

```text
dp[i]
=
max(
    nums[i],
    dp[i-1] + nums[i]
)
```


```text
dp[i]

=
max(
    自己重新开始,
    接上之前最优前缀
)
```

因为：

```text
所有以 i-1 结尾的连续子数组

已经被 dp[i-1]

压缩成一个最优代表。
```

这就是 Kadane Algorithm 的本质。



意思：

```text
要么从自己重新开始

要么接上前面的最大连续和
```

# 四、动态规划公式

## 状态定义

```python
dp[i]
```

表示：

```text
以 nums[i] 结尾的最大连续子数组和
```

---

## 状态转移

```python
dp[i] = max(nums[i], dp[i - 1] + nums[i])
```

解释：

- `nums[i]`：从当前位置重新开始。
- `dp[i - 1] + nums[i]`：接上前面最优连续子数组。

因此：

```python
dp[i] = max(
    nums[i],
    dp[i - 1] + nums[i]
)
```

---

## 最终答案

```python
max(dp)
dp[] 将会存储每个位置下的最优的子数组和
```
```text
比如：
最终：dp = [-2,1,-2,4,3,5,6,1,5]
最大值：6
答案：6
```

# 六、LeetCode核心代码模式

这是面试最常写的版本。

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:

        cur_sum = nums[0]
        max_sum = nums[0]

        for i in range(1, len(nums)):

            cur_sum = max(
                nums[i],
                cur_sum + nums[i]
            )

            max_sum = max(max_sum, cur_sum)

        return max_sum
```

```text
1. cur_sum 表示什么？
cur_sum 表示：
以当前位置 i 结尾的最大连续子数组和
即：cur_sum = dp[i]对应动态规划中的：dp[i]

2. max_sum 表示什么？
max_sum表示：到目前为止出现过的最大子数组和 即：
max(
dp[0],
dp[1],
...
dp[i]
)

cur_sum表示：当前状态
即：必须以当前位置结尾 
以当前位置结尾的最大连续子数组和（当前状态）

max_sum表示：历史最优答案
即：整个数组看到现在的最大值
遍历过程中出现过的最大连续子数组和（最终答案）

必须先求出当前位置的最优状态，再用它更新全局最优答案。

```

# 十、空间优化思想

观察公式：


```python
dp[i] = max(nums[i], dp[i - 1] + nums[i])
```

可以发现：

```text
dp[i]
只依赖
dp[i-1]
```

---

因此：

```python
dp数组
```

根本没必要保存。

只保存：

```python
cur_sum
```

即可。

---

优化后：

```text
时间复杂度：O(n)

空间复杂度：O(1)
```

# 十一、为什么叫 Kadane 算法？

这是经典的：

```text
Kadane's Algorithm
```

思想：

```text
如果当前连续和已经拖后腿了

那就从当前位置重新开始
```

即：

```python
cur_sum = max(
    nums[i],
    cur_sum + nums[i]
)
```
