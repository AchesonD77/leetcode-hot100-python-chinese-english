# 一、题目描述

给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置。

要求：

```text
尽可能使用 O(1) 额外空间完成。
```

---

## 示例 1

```text
输入：

nums = [1,2,3,4,5,6,7]
k = 3

输出：

[5,6,7,1,2,3,4]
```

解释：

```text
向右轮转1次：

[7,1,2,3,4,5,6]

向右轮转2次：

[6,7,1,2,3,4,5]

向右轮转3次：

[5,6,7,1,2,3,4]
```

---

## 示例 2

```text
输入：

nums = [-1,-100,3,99]
k = 2

输出：

[3,99,-1,-100]
```

# 二、题目本质

本题本质是：

```text
把数组最后 k 个元素

移动到数组最前面
```

例如：

```python
nums = [1,2,3,4,5,6,7]
k = 3
```

实际上变成：

```python
[5,6,7] + [1,2,3,4]
```

得到：

```python
[5,6,7,1,2,3,4]
```

---

# 三、面试中的三种解法

本题面试经常考：

```text
方法1：额外数组（最容易）

方法2：切片（Python最快写法）

方法3：三次翻转（最优解）
```

# 四、方法1：额外数组

---

## 思路

创建新数组：

```python
new_nums = [0] * n
```

对于每个元素：

```python
nums[i]
```

计算它的新位置：

```python
(i + k) % n
```

---

## 为什么是这样？

例如：

```python
nums = [1,2,3,4,5]
```

长度：

```python
n = 5
```

右移：

```python
k = 2
```

---

元素：

```python
1
```

原位置：

```python
0
```

新位置：

```python
(0+2)%5 = 2
```

---

元素：

```python
4
```

原位置：

```python
3
```

新位置：

```python
(3+2)%5

= 5%5

= 0
```

成功回到开头。

---

## LeetCode代码

```python
class Solution:
    def rotate(self, nums: List[int], k: int) -> None:

        n = len(nums)

        k %= n

        new_nums = [0] * n

        for i in range(n):
            new_nums[(i + k) % n] = nums[i]

        nums[:] = new_nums
```

---

## 时间复杂度

```text
O(n)
```

---

## 空间复杂度

```text
O(n)
```

# 五、方法2：Python切片

这是 Python 面试最常见写法。

---

## 核心思想

例如：

```python
nums = [1,2,3,4,5,6,7]
k = 3
```

最后：

```python
[5,6,7]
```

放前面。

前半部分：

```python
[1,2,3,4]
```

放后面。

---

得到：

```python
[5,6,7] + [1,2,3,4]
```

---

## LeetCode代码

```python
class Solution:
    def rotate(self, nums: List[int], k: int) -> None:

        n = len(nums)

        k %= n

        nums[:] = nums[-k:] + nums[:-k]
```

```text
nums[:]
表示：整个列表（原地修改）
相当于：nums[0:len(nums)]

为什么不是直接写？
很多同学会写：nums = nums[-k:] + nums[:-k]
和nums[:] = nums[-k:] + nums[:-k]
看起来一样。
实际上完全不同。

nums = [5,6,7,1,2,3,4]
注意：这里是：让变量 nums 指向了一个新的列表
原来的列表没变。

nums[:]
表示：整个列表的所有位置
注意：修改的是原来的列表对象 不是创建新列表。

这句代码完整含义
nums[:] = nums[-k:] + nums[:-k]
可以读成：
取出最后 k 个元素放到前面，再把前面的元素接到后面，并把结果覆盖回原数组。
```

---

## 过程分析

```python
nums = [1,2,3,4,5,6,7]
```

```python

```


---

### nums[:-k]

```python

```

得到：

```python
[1,2,3,4]
```

---

拼接：

```python
[5,6,7] + [1,2,3,4]
```

得到：

```python
[5,6,7,1,2,3,4]
```

# 七、为什么三次翻转有效？

目标：

```python
A = [1,2,3,4]
B = [5,6,7]
```

原数组：

```python
A + B
```

---

整体翻转：

```python
reverse(A+B)
```

得到：

```python
reverse(B) + reverse(A)
```

---

翻转前半部分：

```python
B + reverse(A)
```

---

翻转后半部分：

```python
B + A
```

正好完成轮转。

# 八、LeetCode核心代码模式（最优解）

```python
class Solution:

    def rotate(self, nums: List[int], k: int) -> None:

        n = len(nums)

        k %= n

        self.reverse(nums, 0, n - 1)

        self.reverse(nums, 0, k - 1)

        self.reverse(nums, k, n - 1)

    def reverse(self, nums, left, right):

        while left < right:

            nums[left], nums[right] = \
                nums[right], nums[left]

            left += 1
            right -= 1
```

# 十一、ACM模式写法

---

## 输入格式

```text
7
1 2 3 4 5 6 7
3
```

表示：

```python
nums = [1,2,3,4,5,6,7]

k = 3
```

---

## ACM代码（最优解）

```python
def reverse(nums, left, right):

    while left < right:

        nums[left], nums[right] = \
            nums[right], nums[left]

        left += 1
        right -= 1


def solve():

    n = int(input())

    nums = list(map(int, input().split()))

    k = int(input())

    k %= n

    reverse(nums, 0, n - 1)

    reverse(nums, 0, k - 1)

    reverse(nums, k, n - 1)

    print(*nums)


solve()
```

---

## 输入

```text
7
1 2 3 4 5 6 7
3
```

---

## 输出

```text
5 6 7 1 2 3 4
```
