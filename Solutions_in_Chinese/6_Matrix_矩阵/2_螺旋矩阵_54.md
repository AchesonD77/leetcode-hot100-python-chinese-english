# LeetCode 54. 螺旋矩阵（Spiral Matrix）

## 题目描述
![alt text](<../../Snap_for_Questions/6_Matrix_矩阵/截屏2026-06-16 18.11.59.png>)

```python
class Solution:
    def spiralOrder(self, matrix: List[List[int]]) -> List[int]:
        # 变量 用于 接受res
        res = []

        # 变量 设置 四个方向的值 (表下标，所以从0开始)
        left = 0
        right = len(matrix[0])-1
        top = 0
        bottom = len(matrix)-1

        # 判断持续执行的条件，并进行螺旋循环，分为四条方向拆分进行
        while top <= bottom and left <= right:
            # 从 左 -> 右 移动 (使用列数,此处right要+1，因为right为index)，并且注意有起始点为 left（因为要不停循环，所以不能默认0）
            for j in range(left, right+1):
                res.append(matrix[top][j])
            top += 1
        
            # 从上 -> 下 移动，使用行数，此处bottom要+1,因为bottom为index。并且注意有起始点为right（要不停循环，所以不能默认0）
            for i in range(top, bottom+1):
                res.append(matrix[i][right])
            right -= 1

            # 需要进行判断：是否单边触碰了条件，我们移动左右，则是固定了行 top bottom来移动的（列数）
            # 从右 -> 左 移动，使用列数，此处left-1表示0之后的一位，在range中表示left-1+1的位置也就是0
            if top <= bottom:
                for j in range(right, left-1, -1):
                    res.append(matrix[bottom][j])
                bottom -= 1
            
            # 从下 -> 上 移动，使用行数，此处top-1表示0之后的一位，在range中表示top-1+1的位置也就是0
            if left <= right:
                for i in range(bottom, top-1, -1):
                    res.append(matrix[i][left])
                left += 1

        return res
```
给你一个 `m × n` 的矩阵 `matrix`。

请按照 **顺时针螺旋顺序**，返回矩阵中的所有元素。

# 一、题目本质

这是一道经典的 **矩阵模拟（Matrix Simulation）** 题。

我们的目标：

按照下面四个方向不断移动：

```
右 → 下 → 左 → 上
```

每走完一圈，矩阵边界不断缩小。

---

# 二、核心思想：四个边界控制法

使用四个变量表示当前还没有遍历的边界：

```python
top    上边界
bottom 下边界
left   左边界
right  右边界
```

例如：

初始矩阵：

```
      left        right
        ↓           ↓
      0 1 2 3 4
top → ┌─────────┐
      │         │
      │         │
      └─────────┘
bottom
```

初始值：

```python
top = 0
bottom = m - 1

left = 0
right = n - 1
```

---

# 三、遍历顺序（非常重要）

每一层按照固定顺序：

```
① 从左到右
② 从上到下
③ 从右到左
④ 从下到上
```

即：

```
          →
        ↑   ↓
          ←
```

每完成一个方向，就缩小对应边界。

# 四、方向 1：从左到右

例如：

```
1 2 3
```

代码：

```python
for j in range(left, right + 1):
    result.append(matrix[top][j])
```

## 为什么是 `right + 1`？

Python 的 `range()`：

```python
range(start, end)
```

特点：

```text
左闭右开
包含 start
不包含 end
```

例如：

```python
range(0, 3)
```

得到：

```python
0, 1, 2
```

所以想遍历到：

```python
right （我们设置了right = n - 1）
```

必须写：

```python
right + 1
```

---

遍历完成：

```python
top += 1
```

表示：

```
最上面一行已经访问过
```

所以：

```
上边界向下移动
```

# 五、方向 2：从上到下

代码：

```python
for i in range(top, bottom + 1):
    result.append(matrix[i][right])
```

访问：

```
当前最右侧一列
```

结束后：

```python

```

因为：

```

```

所以：

```
向左移动
```

# 六、方向 3：从右到左

代码：

```python

    result.append(matrix[bottom][j])
```

## `range(right, left - 1, -1)` 详细解释

Python 的倒序遍历：

```python
range(start, end, step)
```

其中：

```python
step = -1
```

表示每次减一。

例如：

```python
right = 3
left = 0
```

那么：

```python
range(3, -1, -1)
```

结果：

```python
3, 2, 1, 0
```

因为：

```python
range()
```

依然遵循：

```
左闭右开
```

所以要写：

```python
left - 1
```

才能访问到 `left`。

---

遍历完成：

```python
bottom -= 1
```

表示：

```
底部这一行已经访问完成
```

# 八、什么时候停止？

使用：

```python
while left <= right and top <= bottom:
```

原因：

当：

```python
left > right
```

说明：

```
没有剩余的列
```

或者：

```python
top > bottom
```

说明：

```
没有剩余的行
```

遍历结束。

---

# 九、关键难点：为什么每次都要判断边界？

假设矩阵：

```python
[
 [1, 2, 3, 4]
]
```

只有一行。

完成：

```
从左到右
```

以后：

```python
top = 1
bottom = 0
```

此时：

```
已经没有行了
```

如果继续：

```
从上到下
从右到左
从下到上
```

会导致重复访问。

所以每完成一个方向后，需要检查边界。

很多人会问：

外面不是已经有 while top <= bottom and left <= right 了吗？为什么里面还需要 if top <= bottom 和 if left <= right？

答案是：

因为 while 只保证进入这一圈之前边界有效，但是前两个方向执行完以后，边界可能已经变化，导致后两个方向已经不存在了。

```text
先理解 while 只检查“一开始”

例如：

while top <= bottom and left <= right:

意思：

当前还有至少一行和一列
可以继续处理这一圈

但是注意：

进入 while 后，你会不断改变边界

所以：

while 检查的是进入这一圈之前
if 检查的是执行到当前方向时是否还存在

注意这里访问的是：matrix[bottom][j]
也就是说：固定 bottom 这一行从右往左走

所以必须保证：
bottom 这一行还存在
判断方式：
top <= bottom

同理：
注意这里访问：matrix[i][left]
说明：固定 left 这一列 从下往上走
所以必须保证：left 这一列还存在
判断：
left <= right
```

# 十、LeetCode 核心代码模式（可以忽略直接看下一个推荐代码）

```python
class Solution:
    def spiralOrder(self, matrix):
        result = []

        m = len(matrix)
        n = len(matrix[0])

        top = 0
        bottom = m - 1

        left = 0
        right = n - 1


        while left <= right and top <= bottom:

            # 1. 从左到右
            for j in range(left, right + 1):
                result.append(matrix[top][j])

            top += 1


            # 2. 从上到下
            for i in range(top, bottom + 1):
                result.append(matrix[i][right])

            right -= 1


            # 判断是否还有剩余区域
            if not (left <= right and top <= bottom):
                break


            # 3. 从右到左
            for j in range(right, left - 1, -1):
                result.append(matrix[bottom][j])

            bottom -= 1


            # 4. 从下到上
            for i in range(bottom, top - 1, -1):
                result.append(matrix[i][left])

            left += 1


        return result
```

# 十一、更加标准的面试模板（推荐）

在面试中，更常见的写法是在每个方向后判断边界：

```python
class Solution:
    def spiralOrder(self, matrix):
        res = []

        top = 0
        bottom = len(matrix) - 1

        left = 0
        right = len(matrix[0]) - 1


        while top <= bottom and left <= right:

            # 左 -> 右
            for j in range(left, right + 1):
                res.append(matrix[top][j])

            top += 1


            # 上 -> 下
            for i in range(top, bottom + 1):
                res.append(matrix[i][right])

            right -= 1


            # 右 -> 左
            if top <= bottom:
                for j in range(right, left - 1, -1):
                    res.append(matrix[bottom][j])

                bottom -= 1


            # 下 -> 上
            if left <= right:
                for i in range(bottom, top - 1, -1):
                    res.append(matrix[i][left])

                left += 1


        return res
```

## 3. 四个方向的 for 循环
我们需要知道：
while = 控制一圈一圈
for = 控制一条边一条边

这两个循环负责的层次完全不同。
### 左到右

```python

```

固定行：

```python
top
```

改变列：

```python
j
```

---

### 上到下

```python
matrix[i][right]
```

固定列：

```python
right
```

改变行：

```python
i
```

---

### 右到左

我们走的是“bottom这一行”

代码：matrix[bottom][j]
这里固定的是：
bottom
也就是：
bottom这一整行

```python
matrix[bottom][j]
```

固定：

```python
bottom
```

列递减。


---

### 下到上

```python
matrix[i][left]
```

固定：

```python
left
```

行递减。

---
### 记忆：
```text
四个方向的固定规律（一定要背）
方向	固定什么	变化什么	代码
左 → 右	top 行	j 列	matrix[top][j]
上 → 下	right 列	i 行	matrix[i][right]
右 → 左	bottom 行	j 列	matrix[bottom][j]
下 → 上	left 列	i 行	matrix[i][left]
```

```python
def spiralOrder(matrix):

    if not matrix:
        return []


    res = []

    top = 0
    bottom = len(matrix) - 1

    left = 0
    right = len(matrix[0]) - 1


    while top <= bottom and left <= right:

        for j in range(left, right + 1):
            res.append(matrix[top][j])

        for i in range(top, bottom + 1):
```
