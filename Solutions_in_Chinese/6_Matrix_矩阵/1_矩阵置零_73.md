# LeetCode 73. 矩阵置零（Set Matrix Zeroes）

## 题目描述
![alt text](<../../Snap_for_Questions/6_Matrix_矩阵/截屏2026-06-15 14.22.34.png>)

```python
class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        """
        Do not return anything, modify matrix in-place instead.
        """
        # 我们需要先拿到行和列，也就是二维数组的长度
        m = len(matrix) # 行
        n = len(matrix[0]) # 列

        # 特殊情况为第0行，第0列的情况，单独讨论
        first_row_0 = any(matrix[0][j] == 0 for j in range(n))
        first_col_0 = any(matrix[i][0] == 0 for i in range(m))

        # 开始循环判断 第一行，第一列开始的为0的标记
        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][j] == 0:
                    matrix[i][0] = 0
                    matrix[0][j] = 0
        # 对于有标记的行 进行整行替换为0
        for i in range(1, m):
            if matrix[i][0] == 0:
                for j in range(1, n):
                    matrix[i][j] = 0
        # 对于有标记的列 进行整列替换为0
        for j in range(1, n):
            if matrix[0][j] == 0:
                for i in range(1, m):
                    matrix[i][j] = 0

        # 对于0行0列单独判断 的 替换步骤
        if first_row_0:
            for j in range(n):
                matrix[0][j] = 0

        if first_col_0:
            for i in range(m):
                matrix[i][0] = 0


```

给定一个 `m × n` 的矩阵 `matrix`。

如果一个元素为 `0`，则将其所在行和所在列的所有元素都设为 `0`。

要求：

- 必须 **原地（In-place）修改矩阵**
- 尽量减少额外空间复杂度

# 一、题目本质

当发现一个位置：

```python
matrix[i][j] == 0
```

需要：

```python
第 i 行全部变 0
第 j 列全部变 0
```

---

例如：

```python
1 1 1
1 0 1
1 1 1
```

发现：

```python
matrix[1][1] = 0
```

那么：

```python
第1行全部置0
第1列全部置0
```

结果：

```python
1 0 1
0 0 0
1 0 1
```

---

# 二、最容易想到的方法（暴力法）

## 思路

先记录：

- 哪些行需要变 0
- 哪些列需要变 0

然后第二次遍历矩阵完成修改。

---

## ACM写法

```python
def setZeroes(matrix):
    m = len(matrix)
    n = len(matrix[0])

    rows = set()
    cols = set()

    for i in range(m):
        for j in range(n):
            if matrix[i][j] == 0:
                rows.add(i)
                cols.add(j)

    for i in range(m):
        for j in range(n):
            if i in rows or j in cols:
                matrix[i][j] = 0
```

---

## 时间复杂度

```python
O(m*n)
```

---

## 空间复杂度

```python
O(m+n)
```

因为：

```python

cols
```

额外存储了行和列。

    def setZeroes(self, matrix):
        m = len(matrix)
        n = len(matrix[0])

        first_row_zero = any(matrix[0][j] == 0 for j in range(n))

# 十、ACM模式（牛客/机试模式）

```python
def setZeroes(matrix):
    m = len(matrix)
    n = len(matrix[0])

    first_row_zero = any(matrix[0][j] == 0 for j in range(n))
    first_col_zero = any(matrix[i][0] == 0 for i in range(m))

    for i in range(1, m):
        for j in range(1, n):
            if matrix[i][j] == 0:
                matrix[i][0] = 0
                matrix[0][j] = 0

    for i in range(1, m):
        if matrix[i][0] == 0:
            for j in range(1, n):
                matrix[i][j] = 0

    for j in range(1, n):
        if matrix[0][j] == 0:
            for i in range(1, m):
                matrix[i][j] = 0

    if first_row_zero:
        for j in range(n):
            matrix[0][j] = 0

    if first_col_zero:
        for i in range(m):
            matrix[i][0] = 0

    return matrix


matrix = [
    [1,1,1],
    [1,0,1],
    [1,1,1]
]

print(setZeroes(matrix))
```
