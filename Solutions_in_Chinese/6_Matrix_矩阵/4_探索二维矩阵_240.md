# LeetCode 240. 搜索二维矩阵 II（Search a 2D Matrix II）

## 题目描述

给定一个 `m × n` 的二维矩阵 `matrix` 和一个目标值 `target`。

矩阵具有如下特点：

1. 每一行的元素从左到右升序排列。
2. 每一列的元素从上到下升序排列。

请设计一个高效算法，判断目标值 `target` 是否存在于矩阵中。

存在返回：

```python
True
```

不存在返回：

```python
False
```

---

输入：

```python
matrix = [
    [1,4,7,11,15],

    [3,6,9,16,22],
    [10,13,14,17,24],
    [18,21,23,26,30]
]

target = 5
```

输出：

```python
True
```

# 三、核心思想：从右上角开始搜索（最优）

## 为什么选择右上角？

假设：

```
        列 →
        0   1   2   3   4
      +-----------------
行 0  | 1   4   7  11  15 ← 起点
行 1  | 2   5   8  12  19
行 2  | 3   6   9  16  22
行 3  |10  13  14  17  24
行 4  |18  21  23  26  30
      ↓
```

选择：

```python
matrix[0][n - 1]
```

即右上角。

原因：

右上角有两个特点：

---

## 1. 左边元素更小

例如：

```
1 4 7 11 15
          ↑
```

左边：

```text
都小于 15
```

---

## 2. 下方元素更大

例如：

```
15
19
22
24
30
```

下面：

```text
都大于 15
```

因此可以每次排除一整行或一整列。


如果：




# 六、LeetCode 核心代码模式（推荐背诵）

```python
class Solution:
    def searchMatrix(self, matrix, target):

        # 获取矩阵大小
        m = len(matrix)
        n = len(matrix[0])

        # 从右上角开始
        row = 0
        col = n - 1


        # 只要没有越界继续搜索
        while row < m and col >= 0:

            current = matrix[row][col]


            # 找到目标
            if current == target:
                return True


            # 当前值太大，向左移动
            elif current > target:
                col -= 1


            # 当前值太小，向下移动
            else:
                row += 1


        # 搜索结束仍未找到
        return False
```

# 八、ACM 模式（牛客 / 机试）

```python
def searchMatrix(matrix, target):

    if not matrix or not matrix[0]:
        return False


    m = len(matrix)
    n = len(matrix[0])


    row = 0
    col = n - 1


    while row < m and col >= 0:

        if matrix[row][col] == target:
            return True

        elif matrix[row][col] > target:
            col -= 1

        else:
            row += 1


    return False


matrix = [
    [1,4,7,11,15],
    [2,5,8,12,19],
    [3,6,9,16,22],
    [10,13,14,17,24],
    [18,21,23,26,30]
]


target = 5


print(searchMatrix(matrix, target))
```

输出：

```python
True
```

# 十、为什么叫“二维矩阵中的双指针”？

虽然没有传统数组中的：

```text
left
right
```

但是：

```python
row
col
```

实际上也是两个移动指针：

```
        col ←
        ↓
1 4 7 11 15
          ↑
          row
```

移动规则：

```text
太大：col--
太小：row++
```

因此本题也属于：

```text
双指针 / 双变量移动
```

# 十一、复杂度分析

矩阵大小：

```text
m × n
```

---

## 时间复杂度

每次：

- 要么左移一列
- 要么下移一行

最多移动：

```text
m + n 次
```

所以：

```text
时间复杂度：O(m + n)
```

---

## 空间复杂度

只使用：

```text
row
col
current
```

几个变量。

所以：

```text
空间复杂度：O(1)
```
