# 中文版

## 一、题目核心

给定一个字符串 `s`：

> 找到“不包含重复字符”的最长子串长度。

注意：

- 子串（substring）必须连续
- 子序列（subsequence）不要求连续

示例：

```text
输入：
s = "abcabcbb"

输出：
3

最长无重复子串：
"abc"
```

---

## 二、核心思路（滑动窗口）

本题核心：

```text
滑动窗口 + HashSet
```

窗口中始终保持：

```text
没有重复字符
```

如果出现重复：

```text
不断移动 left 指针
直到窗口重新合法
```

---

## 三、动画理解

字符串：

```text
s = "abcabcbb"
```

窗口变化：

```text
[a]      -> 1
[ab]     -> 2
[abc]    -> 3
[abca]   -> 出现重复 a
```

此时：

```text
删除左边的 a
```

窗口变成：

```text
[bca]
```

继续向右扩展。

---

# 四、Python ACM 模式代码（面试常用）

```python
# ACM 模式

s = input()

char_set = set()

left = 0
max_len = 0

for right in range(len(s)):

    # 出现重复字符
    while s[right] in char_set:
        char_set.remove(s[left])
        left += 1

    # 加入当前字符
    char_set.add(s[right])

    # 更新最大长度
    max_len = max(max_len, right - left + 1)

print(max_len)
```


作用：

## 6. 加入当前字符

```python
char_set.add(s[right])
```

表示：


# 六、必须掌握的 Python 语法（算法核心）

创建：

```python

```

添加：

```python
s.add("a")
```

删除：

```python
s.remove("a")
```

特点：

滑动窗口高频使用。

## 2. while 循环

```python
while 条件:
```

本题：

```python
while s[right] in char_set:
```

含义：

```text
只要重复
就一直缩小窗口
```

这是滑动窗口核心思想。

## 3. 双指针

```python
left
right
```

作用：

| 指针 | 含义 |
|---|---|
| left | 缩小窗口 |
| right | 扩大窗口 |

本质：

```text
维护一个动态区间
```

---

## 4. max()

```python
max(a, b)
```

作用：

```text
取最大值
```

本题：

```python
max_len = max(max_len, 当前窗口长度)
```

---

## 5. 窗口长度公式

```python
right - left + 1
```

例如：

```text
left = 2
right = 4
```

实际长度：

```text
3
```

因为：

```text
4 - 2 + 1 = 3
```

# 七、时间复杂度

## 时间复杂度

```text
O(n)
```

原因：

每个字符：

- 最多进入窗口一次
- 最多离开窗口一次

所以总复杂度：

```text
O(n)
```

## 空间复杂度

```text
O(n)
```

# 八、面试必须会说的话

```text
本题使用滑动窗口 + HashSet。

right 指针负责扩大窗口，
left 指针负责缩小窗口。

如果发现重复字符，
```
