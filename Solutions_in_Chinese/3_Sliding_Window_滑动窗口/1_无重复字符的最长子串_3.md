# 中文版

## 一、题目核心

给定一个字符串 `s`：

> 找到“不包含重复字符”的最长子串长度。

注意：

- 子串（substring）必须连续
- 子序列（subsequence）不要求连续

示例：

```text
输入：
s = "abcabcbb"

输出：
3

最长无重复子串：
"abc"
```

---

## 二、核心思路（滑动窗口）

本题核心：

```text
滑动窗口 + HashSet
```

窗口中始终保持：

```text
没有重复字符
```

如果出现重复：

```text
不断移动 left 指针
直到窗口重新合法
```

---

## 三、动画理解

字符串：

```text
s = "abcabcbb"
```

窗口变化：

```text
[a]      -> 1
[ab]     -> 2
[abc]    -> 3
[abca]   -> 出现重复 a
```

此时：

```text
删除左边的 a
```

窗口变成：

```text
[bca]
```

继续向右扩展。

# 四、Python ACM 模式代码（面试常用）

```python
# ACM 模式

s = input()

char_set = set()

left = 0
max_len = 0

for right in range(len(s)):

    # 出现重复字符
    while s[right] in char_set:
        char_set.remove(s[left])
        left += 1

    # 加入当前字符
    char_set.add(s[right])

    # 更新最大长度
    max_len = max(max_len, right - left + 1)

print(max_len)
```

# 滑动窗口常见疑问总结

## 问题 1

### 问：

```text
字符串 "abb" 中，当 right 指向第二个 b 时，
remove(s[left]) 先删除了 a，不是保留了两个 b 吗？
```

### 答：

```text
不会，因为 while 会持续删除左边字符，直到重复字符消失为止。
```

---

## 问题 2

### 问：

```text
"abb" 最后窗口只剩一个 b，
那最长无重复子串长度不就变成 1 了吗？
```

### 答：

```text
不会，因为 max_len 记录的是历史最大窗口长度，而不是最终窗口长度。
```

# 七、时间复杂度

## 时间复杂度

```text
O(n)
```

原因：

每个字符：

- 最多进入窗口一次
- 最多离开窗口一次

所以总复杂度：

```text
O(n)
```

---

## 空间复杂度

```text
O(n)
```

最坏情况：

```text
所有字符都不同
```

# 八、面试必须会说的话

```text
本题使用滑动窗口 + HashSet。

right 指针负责扩大窗口，
left 指针负责缩小窗口。

窗口内始终保持没有重复字符。

如果发现重复字符，
就移动 left 指针，
直到窗口重新合法。

时间复杂度为 O(n)。
```
