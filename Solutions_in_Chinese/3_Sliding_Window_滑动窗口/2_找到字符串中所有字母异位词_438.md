# Find All Anagrams in a String

## 一、题目核心

给定两个字符串：

```text
s = "cbaebabacd"
p = "abc"
```

找出：

```text
s 中所有是 p 异位词的子串起始位置
（异位词：表示字母一样，位置不一样）
```

返回：

```text
[0, 6]
```

因为：

```text
s[0:3] = "cba"
s[6:9] = "bac"
```

都可以重新排列成：

```text
"abc"
```

---

## 二、什么是异位词（Anagram）

异位词：

```text
字符种类相同
字符数量相同
字符顺序可以不同
```

例如：

```text
abc
acb
bac
bca
cab
cba
```

都是异位词。

---

## 三、核心思路（固定长度滑动窗口）

观察：

```text
p = "abc"
长度 = 3
```

那么：

```text
s 中所有长度为 3 的子串
都需要检查
```

例如：

```text
cba
bae
aeb
eba
bab
aba
bac
acd
```

---

窗口长度始终固定：

```text
len(p)
```

每次：

```text
右边加入一个字符
左边删除一个字符
```

比较：

```text
窗口字符统计
==
p字符统计
```

如果相同：

```text
找到一个异位词
```

## 四、Python ACM 模式代码（面试常用）

```python
from collections import Counter

s = input().strip()
p = input().strip()

need = Counter(p)
window = Counter()

left = 0
result = []

for right in range(len(s)):

    window[s[right]] += 1

    if right - left + 1 > len(p):
        window[s[left]] -= 1

        if window[s[left]] == 0:
            del window[s[left]]

        left += 1

    if right - left + 1 == len(p):
        if window == need:
            result.append(left)

print(result)
```

## 四、LeetCode 模式
```python
from collections import Counter
from typing import List

class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:

        need = Counter(p)
        window = Counter()

        left = 0
        result = []

        for right in range(len(s)):

            # 右侧加入字符
            window[s[right]] += 1

            # 窗口过长
            if right - left + 1 > len(p):

                window[s[left]] -= 1

                if window[s[left]] == 0:
                    del window[s[left]]

                left += 1

            # 长度符合要求
            if right - left + 1 == len(p):

                if window == need:
                    result.append(left)

        return result
```
其中：window['a'] += 1 表示 把字符a的出现次数加1

window[s[left]] -= 1 是什么？ 窗口左边元素离开窗口。滑动窗口本质：右边进 左边出 就像一个队列。

# 附加讲解：
# 四、为什么 Counter 可以直接 `+= 1`

普通字典：

```python
d = {}
d['a'] += 1
```

会报错：

```python
KeyError
```

因为：

```python
'a'
```

这个 Key 还不存在。

---

但是：

```python
from collections import Counter

window = Counter()
```

`Counter` 是 Python 提供的特殊哈希表。

对于不存在的 Key：

```python
window['a']
```

默认返回：

```python
0
```

而不是报错。

所以：

```python
window['a'] += 1
```

实际上相当于：

```python
window['a'] = 0 + 1
```

结果：

```python
Counter({'a': 1})
```

---

## Counter 本质

```python
window = Counter()
```

可以理解成：

```python
window = {
    字符: 出现次数
}
```

例如：

```python
Counter({
    'a': 2,
    'b': 1
})
```

表示：

```text
a 出现 2 次
b 出现 1 次
```

# 五、`window[s[left]] -= 1` 是什么？

表示：

```text
窗口左边的字符离开窗口
```

所以：

```text
该字符出现次数减 1
```

---

例如：

当前窗口统计：

```python
window = {
    'a': 2,
    'b': 1
}
```

左侧字符：

```python
s[left] = 'a'
```

执行：

```python
window['a'] -= 1
```

结果：

```python
{
    'a': 1,
    'b': 1
}
```

意思：

```text
窗口中少了一个 a
```

---

## 本质理解

```python
window[ch] += 1
```

表示：

```text
字符进入窗口
次数 +1
```

---

```python
window[ch] -= 1
```

表示：

```text
字符离开窗口
次数 -1
```

# 六、为什么要减 1？

因为滑动窗口本质上是：

```text
右边进
左边出
```

就像一个移动的盒子。

---

例如：

```python
s = "cbaebabacd"
p = "abc"
```

窗口长度：

```python
len(p) = 3
```

---

初始窗口：

```text
[c b a]
```

统计：

```python
{
    'c':1,
    'b':1,
    'a':1
}
```

---

窗口向右滑动：

```text
c [b a e]
```

发生了两件事：

### ① c 离开窗口

```python
window['c'] -= 1
```

结果：

```python
{
    'c':0,
    'b':1,
    'a':1
}
```

---

### ② e 进入窗口

```python
window['e'] += 1
```

结果：

```python
{
    'c':0,
    'b':1,
    'a':1,
    'e':1
}
```

---

## 滑动窗口核心思想

每向右移动一次：

```text
左边字符删除
右边字符加入
```

即：

```python
window[s[left]] -= 1
window[s[right]] += 1
```

# 七、为什么还要删除 0？

代码：

```python
if window[s[left]] == 0:
    del window[s[left]]
```

---

例如：

当前：

```python
{
    'c':0,
    'b':1,
    'a':1,
    'e':1
}
```

删除后：

```python
{
    'b':1,
    'a':1,
    'e':1
}
```

---

## 原因：后面需要比较两个 Counter

代码：

```python
if window == need:
```

---

假设：

```python
need = {
    'a':1,
    'b':1,
    'e':1
}
```

---

如果保留：

```python
window = {
    'a':1,
    'b':1,
    'e':1,
    'c':0
}
```

和：

```python
need = {
    'a':1,
    'b':1,
    'e':1
}
```

比较：

```python
window == need
```

结果：

```python
False
```

因为：

```text
Key 不完全相同
```

---

## 删除 0 后

```python
window = {
    'a':1,
    'b':1,
    'e':1
}
```

此时：

```python
window == need
```

结果：

```python
True
```
