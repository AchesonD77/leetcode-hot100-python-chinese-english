# LeetCode Hot100 — 438. 找到字符串中所有字母异位词
# Find All Anagrams in a String

# 中文版

## 一、题目核心

给定两个字符串：

```text
s = "cbaebabacd"
p = "abc"
```

找出：

```text
s 中所有是 p 异位词的子串起始位置
```

返回：

```text
[0, 6]
```

因为：

```text
s[0:3] = "cba"
s[6:9] = "bac"
```

都可以重新排列成：

```text
"abc"
```

---

## 二、什么是异位词（Anagram）

异位词：

```text
字符种类相同
字符数量相同
字符顺序可以不同
```

例如：

```text
abc
acb
bac
bca
cab
cba
```

都是异位词。

---

## 三、核心思路（固定长度滑动窗口）

观察：

```text
p = "abc"
长度 = 3
```

那么：

```text
s 中所有长度为 3 的子串
都需要检查
```

例如：

```text
cba
bae
aeb
eba
bab
aba
bac
acd
```

---

窗口长度始终固定：

```text
len(p)
```

每次：

```text
右边加入一个字符
左边删除一个字符
```

比较：

```text
窗口字符统计
==
p字符统计
```

如果相同：

```text
找到一个异位词
```

---

## 四、Python ACM 模式代码（面试常用）

```python
from collections import Counter

s = input().strip()
p = input().strip()

need = Counter(p)
window = Counter()

left = 0
result = []

for right in range(len(s)):

    window[s[right]] += 1

    if right - left + 1 > len(p):
        window[s[left]] -= 1

        if window[s[left]] == 0:
            del window[s[left]]

        left += 1

    if right - left + 1 == len(p):
        if window == need:
            result.append(left)

print(result)
```


# 九、面试必须会说的话

```text
本题使用固定长度滑动窗口。

窗口长度始终等于 len(p)。

使用 Counter 统计目标字符串和当前窗口字符频率。

每次窗口右移时：
右边加入一个字符，
左边删除一个字符。

如果两个频率表相同，
说明当前窗口是一个异位词。

时间复杂度 O(n)，
空间复杂度 O(1)。
```
