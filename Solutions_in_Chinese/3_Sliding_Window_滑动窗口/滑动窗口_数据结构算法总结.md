# LeetCode Hot100 滑动窗口专题（Sliding Window）
# 一篇彻底搞懂滑动窗口

# 一、什么是滑动窗口？

滑动窗口（Sliding Window）本质上是：

```text
双指针维护一个连续区间
```

通常使用：

```python
left = 0
right = 0
```

表示窗口：

```text
[left, right]
```

例如：

```text
s = "abcdef"

窗口：

[a]
[ab]
[abc]
[abcd]
```

随着 right 向右移动：

```text
窗口不断扩大
```

随着 left 向右移动：

```text
窗口不断缩小
```

---

# 二、为什么要用滑动窗口？

暴力做法：

```text
枚举所有子串
```

时间复杂度：

```text
O(n²)
甚至 O(n³)
```

而滑动窗口：

```text
每个元素最多进窗口一次
每个元素最多出窗口一次
```

因此：

```text
时间复杂度 O(n)
```

---

# 三、滑动窗口的本质

牢记一句话：

```text
right 负责扩张窗口

left 负责缩小窗口
```

流程：

```text
① right向右扩张

② 判断是否合法

③ 不合法：
    left缩小窗口

④ 合法：
    更新答案
```

模板：

```python
for right in range(len(s)):

    # 右边加入元素

    while 不满足条件:
        # 左边删除元素
        left += 1

    # 更新答案
```

---

# 四、滑动窗口分类（最重要）

滑动窗口只分两类：

---

## 类型1：可变长度窗口

窗口长度不固定。

例如：

```text
3. 无重复字符的最长子串

76. 最小覆盖子串

209. 长度最小的子数组
```

特点：

```text
窗口长度不断变化
```

模板：

```python
for right in range(len(s)):

    # 扩张窗口

    while 不合法:
        left += 1

    更新答案
```

---

## 类型2：固定长度窗口

窗口长度固定。

例如：

```text
438. 找到所有字母异位词

567. 字符串的排列
```

特点：

```text
窗口长度始终固定
```

模板：

```python
for right in range(len(s)):

    # 加入右边元素

    if 窗口过长:
        删除左边元素

    if 窗口长度满足:
        判断答案
```

---

# 五、滑动窗口万能模板

## 模板一：可变长度窗口

```python
left = 0

for right in range(len(s)):

    # 加入右边元素

    while 不满足条件:
        # 删除左边元素
        left += 1

    # 更新答案
```

适用于：

```text
最长
最短
最多
最少
满足条件
```

类型题目。

---

## 模板二：固定长度窗口

```python
left = 0

for right in range(len(s)):

    # 加入右边元素

    if right - left + 1 > k:

        # 删除左边元素
        left += 1

    if right - left + 1 == k:

        # 判断答案
```

适用于：

```text
长度固定为 k
```

类型题目。

# 九、面试必须会说的话

```text
滑动窗口本质是双指针维护一个连续区间。

right负责扩张窗口，
left负责缩小窗口。

每个元素最多进入窗口一次，
最多离开窗口一次。

因此时间复杂度通常为 O(n)。

固定窗口用于固定长度问题；

可变窗口用于最长、最短、满足条件类问题。
```
