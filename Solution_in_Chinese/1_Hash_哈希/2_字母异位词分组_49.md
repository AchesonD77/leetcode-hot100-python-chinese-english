# 49. Group Anagrams（字母异位词分组）

![alt text](<../../Snap_for_Questions/1_Hash_哈希/chinese_version/截屏2026-05-22 15.44.40.png>)
## 题目核心

给定一个字符串数组，把：

```text
字母相同，但顺序不同的单词
```

放到同一个组中。

---

# 什么是字母异位词？

例如：

```python
"eat"
"tea"
"ate"
```

它们包含：

```text
相同的字母
```

只是顺序不同。

---

# 核心思想

## 排序后作为哈希表 key

因为：

```python
"eat" -> "aet"
"tea" -> "aet"
"ate" -> "aet"
```

排序后都相同。

所以：

```text
排序后的字符串
```

可以作为：

```text
哈希表 key
```

---

# Python 最终代码

```python
from collections import defaultdict

class Solution:
    def groupAnagrams(self, strs):

        hashmap = defaultdict(list)

        for s in strs:

            key = "".join(sorted(s))

            hashmap[key].append(s)

        return list(hashmap.values())
```

---

# 代码讲解

## 1. 创建哈希表

```python
hashmap = defaultdict(list)
```

作用：

```text
自动创建空列表
```

例如：

```python
hashmap["aet"] = []
```

---

## 2. 遍历字符串

```python
for s in strs:
```

例如：

```python
s = "eat"
```

---

## 3. 排序字符串（核心）

```python
key = "".join(sorted(s))
```

例如：

```python
sorted("eat")
```

结果：

```python
['a', 'e', 't']
```

再通过：

```python
"".join(...)
```

变成：

```python
"aet"
```

---

## 4. 放入对应组

```python
hashmap[key].append(s)
```

例如：

```python
hashmap["aet"] = ["eat", "tea", "ate"]
```

---

## 5. 返回结果

```python
return list(hashmap.values())
```

### list()

```python
list()
```

是 Python 内置函数，用来把可遍历对象转换成列表。


返回：

```python
[
    ["eat","tea","ate"],
    ["tan","nat"],
    ["bat"]
]
```
### 二维数组（2D List）

```python
[[], [], []]
```

这种结构叫：

```text
二维数组 / 嵌套列表（List of Lists）
```

不是字典。

# 时间复杂度

$$
O(n \cdot k \log k)
$$

其中：

```text
n = 字符串数量
k = 字符串平均长度
```

因为每个字符串都需要排序。

---

# 空间复杂度

$$
O(nk)
$$

因为哈希表需要存储所有字符串。

# 本题必须掌握的 Python 语法

---

# 1. sorted() 排序

```python
sorted(s)
```

作用：

```text
对字符串中的字符进行排序
```

例如：

```python
sorted("eat")
```

结果：

```python
['a', 'e', 't']
```

注意：

```text
sorted() 返回的是列表 list
不是字符串 string
```

所以：

```python
sorted("eat") != "aet"
```

而是：

```python
['a', 'e', 't']
```

---

# 2. "".join() 字符拼接

```python
"".join(sorted(s))
```

作用：

```text
把列表中的字符拼接成字符串
```

例如：

```python
"".join(['a', 'e', 't'])
```

结果：

```python
"aet"
```

这是本题核心步骤：

```python
"eat"
↓
['a', 'e', 't']
↓
"aet"
```

---

# 3. defaultdict(list)

defaultdict(list) 的作用是：当字典中某个 key 不存在时，自动创建一个空列表 []，这样就能直接使用 .append() 添加元素而不会报错。

```python
from collections import defaultdict
```

```python
hashmap = defaultdict(list)
```

作用：

```text
当 key 不存在时
自动创建空列表 []
```

例如：

```python
hashmap["aet"].append("eat")
```

即使：

```python
"aet"
```

之前不存在，也不会报错。

会自动变成：

```python
hashmap["aet"] = []
```

然后再：

```python
append("eat")
```

最终：

```python
{
    "aet": ["eat"]
}
```

---

# 4. append()

```python
list.append(x)
```

作用：

```text
向列表末尾添加元素
```

例如：

```python
arr = []

arr.append("eat")
arr.append("tea")
```

结果：

```python
["eat", "tea"]
```

---

# 5. list(hashmap.values())

```python
list(hashmap.values())
```

作用：

```text
获取哈希表中所有 value
并转换成列表
```

例如：

```python
{
    "aet": ["eat", "tea"],
    "ant": ["tan", "nat"]
}
```

执行：

```python
list(hashmap.values())
```

结果：

```python
[
    ["eat", "tea"],
    ["tan", "nat"]
]
```

# 本题最核心的一行代码

```python
key = "".join(sorted(s))
```

这是：

```text
排序
+
字符串拼接
```

组合。

真正作用：

```text
把具有相同字母特征的单词
转换成相同 key
```

例如：

```python
"eat" -> "aet"
"tea" -> "aet"
"ate" -> "aet"
```

于是：

```text
它们就会进入同一个组
```
