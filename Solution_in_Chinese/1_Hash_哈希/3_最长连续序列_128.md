# 128. Longest Consecutive Sequence（最长连续序列）

![alt text](<../../Snap_for_Questions/1_Hash_哈希/chinese_version/截屏2026-05-22 17.20.42.png>)
## 题目核心

给定一个无序数组：

```python
nums
```

找到：

```text
最长连续数字序列的长度
```

注意：

```text
连续的是数字
不是数组下标
```

例如：

```python
[100,4,200,1,3,2]
```

连续序列是：

```python
[1,2,3,4]
```

所以答案：

```python
4
```

---

# 核心思想

## 使用 set 快速查找

因为：

```text
set 查询速度非常快
```

时间复杂度：

$$
O(1)
$$

---

# 本题真正核心

对于一个数字：

```python
num
```

如果：

```python
num - 1 不存在
```

说明：

```text
它是连续序列的起点
```

然后不断：

```python
num + 1
num + 2
num + 3
```

向后查找即可。

---

# 为什么这样做？

例如：

```python
[1,2,3,4]
```

只有：

```python
1
```

满足：

```python
0 不存在
```

所以：

```text
1 是序列起点
```

而：

```python
2
3
4
```

都不是起点。

这样可以避免重复计算。

---

# Python 最终代码

```python
class Solution:
    def longestConsecutive(self, nums):

        num_set = set(nums)

        longest = 0

        for num in num_set:

            if num - 1 not in num_set:

                current_num = num
                current_length = 1

                while current_num + 1 in num_set:

                    current_num += 1
                    current_length += 1

                longest = max(longest, current_length)

        return longest
```

# 代码讲解

## 1. 转换成 set

是 Python 内置函数，用来创建集合或把可遍历对象转换成集合（自动去重）

```python
num_set = set(nums)
```

作用：

```text
去重
+
快速查找
```

例如：

```python
nums = [1,2,2,3]
```

变成：

```python
{1,2,3}
```

---

## 2. longest

```python
longest = 0
```

记录：

```text
最长连续序列长度
```

---

## 3. 遍历数字

```python
for num in num_set:
```

遍历每个数字。

---

## 4. 判断是否为起点（核心）

```python
if num - 1 not in num_set:
```

意思：

```text
前一个数字不存在
```

说明：

```text
当前数字是连续序列起点
```

例如：

```python
1 前面没有 0
```

所以：

```python
1 是起点
```

---

## 5. 开始向后扩展

```python
while current_num + 1 in num_set:
```

不断寻找：

```python
2
3
4
5
```

直到连续中断。

---

## 6. 更新最长长度

```python
longest = max(longest, current_length)
```

记录最大值。

# 代码执行流程（最重要），着重注意return的位置

## 1. for 循环

负责：

```text
遍历所有数字
```

```python
for num in num_set:
```

---

## 2. if 判断

负责：

```text
判断当前数字是不是连续序列的起点
```

```python
if num - 1 not in num_set:
```

例如：

```python
1,2,3,4
```

只有：

```python
1
```

满足：

```python
0 不在 set 中
```

所以：

```python
1
```

才是起点。

---

## 3. while 循环

负责：

```text
不断向后统计连续长度
```

```python
while current_num + 1 in num_set:
```

例如：

```python
1 -> 2 -> 3 -> 4
```

每次：

```python
current_num += 1
current_length += 1
```

---

## 4. longest = max(...)

负责：

```text
更新全局最长值
```

```python
longest = max(longest, current_length)
```

意思：

```text
比较：
当前最长值
和
当前连续序列长度

保留更大的那个
```

---

## 5. return longest

负责：

```text
最终返回答案
```

```python
return longest
```

必须：

```text
放在整个 for 循环结束之后
```

因为：

```text
需要先检查完所有数字，
才能得到全局最长连续序列长度。
```
