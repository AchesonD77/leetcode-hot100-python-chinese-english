# 中文版本

## 一、题目核心

给定数组：

```python
nums = [-1,0,1,2,-1,-4]
```

找出所有：

```text
三个数之和 = 0
```

的组合。

要求：

- 不能重复
- 返回所有答案

---

## 二、核心思想（排序 + 双指针）

步骤：

### 1. 先排序

```python
nums.sort()
```

排序后：

```python
[-4,-1,-1,0,1,2]
```

---

### 2. 固定一个数

例如：

```python
nums[i]
```

然后：

- 左边一个指针
- 右边一个指针

去寻找：

```text
nums[left] + nums[right] = -nums[i]
```

---

## 三、为什么一定要排序（重点）

排序后才能：

- 使用双指针
- 快速判断该移动哪边
- 去重

这是本题核心。三个位置任何一个重复，
都可能产生重复答案。所有双指针+固定的位置 都要去重

---

## 四、Python 代码（必须背会）

```python
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        nums.sort()

        result = []

        n = len(nums)

        for i in range(n):

            # 去重
            if i > 0 and nums[i] == nums[i - 1]:
                continue

            left = i + 1
            right = n - 1

            while left < right:

                total = nums[i] + nums[left] + nums[right]

                if total < 0:
                    left += 1

                elif total > 0:
                    right -= 1

                else:
                    result.append([nums[i], nums[left], nums[right]])

                    # 左去重
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1

                    # 右去重
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1

                    left += 1
                    right -= 1

        return result
```

## 五、代码讲解 | Code Explanation

---

```python
nums.sort()
```

先排序。

作用：

- 双指针必须有序
- 方便去重

---

```python
result = []
```

保存最终答案。

---

```python
n = len(nums)
```

数组长度。

---

```python
for i in range(n):
```

固定第一个数字。

例如：

```python
nums[i]
```

这个数字已经当过“第一个数字”了，
再用一次会产生重复答案，
所以直接跳过。

解析：“为什么重复的第一个数字，
一定会产生重复答案？” Three Sum的本质是在：固定一个数，
再去找另外两个数。如果发现重复数字说明->“固定的第一个数字”
根本没变，这这种情况下：相同的“第一个数字”，
会产生相同的搜索空间（后两个数字一样）。结果一定重复。
---

本质：
这个数字已经作为“起点”
处理过一次了，
没必要再来一次。“同样的开头，
不可能产生新的答案。”

```python
if i > 0 and nums[i] == nums[i - 1]:
    continue
```

第一个数字去重。 这里continue直接结束当前循环，
直接进入下一轮。

避免：

```python
[-1,0,1]
[-1,0,1]
```

重复出现。

---

```python
left = i + 1
right = n - 1
```

双指针：

- `left` 从左边开始
- `right` 从右边开始

---

```python
while left < right:
```

当两个指针没有相遇：

继续寻找答案。

---

```python
total = nums[i] + nums[left] + nums[right]
```

计算三数之和。

---

```python
if total < 0:
    left += 1
```

和太小：

需要更大的数。

因为数组已经排序：

```text
左指针右移
数字会变大
```

---

```python
elif total > 0:
    right -= 1
```

和太大：

需要更小的数。

所以：

```text
右指针左移
数字会变小
```

---

```python
else:
```

找到答案：

```python
total == 0
```

---

```python
result.append([nums[i], nums[left], nums[right]])
```

**append() 一次只能接收一个参数。append() 不一定添加数组，它可以添加任何对象；Three Sum 这里使用 []，是因为每个答案本身就是一个列表。**

保存当前三元组。

---

同样的数字会产生同样的答案，
所以必须全部跳过。
---
while:
    先判断条件
    条件成立 → 执行代码
    执行完 → 回到 while 那一行重新判断，直到：条件不成立 才真正结束。

```python
while left < right and nums[left] == nums[left + 1]:
    left += 1
```

左边去重。

---

```python
while left < right and nums[right] == nums[right - 1]:
    right -= 1
```

右边去重。

---
找到答案后，
当前 left 和 right 都已经用过，
所以两边都必须移动，
才能寻找新的不重复组合。

```python
left += 1
right -= 1
```

继续寻找下一个答案。

---

```python
return result
```

返回最终结果。
