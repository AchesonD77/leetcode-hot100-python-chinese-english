# LeetCode 283. 移动零（Move Zeroes） from Two pointers 双指针

# 中文版本
![alt text](<../../Snap_for_Questions/2_TwoPointers_双指针/chinese_version/截屏2026-05-25 17.08.56.png>)

## 一、题目核心

把数组中的所有 `0` 移动到最后面。

要求：

- 非零元素顺序不变
- 原地修改数组
- 不能新建数组

---

## 二、核心思想（双指针）

使用两个指针：

- `fast`：遍历数组
- `slow`：指向下一个放非零元素的位置

规则：

- 遇到非零：
  - 和 `slow` 交换
  - `slow += 1`
- 遇到 `0`：
  - 跳过

---

## 三、Python 代码（必须背会）

```python
class Solution:
    def moveZeroes(self, nums: list[int]) -> None:
        slow = 0

        for fast in range(len(nums)):
            if nums[fast] != 0:
                nums[slow], nums[fast] = nums[fast], nums[slow]
                slow += 1
```

---

## 五、执行流程（最重要）

输入：

```python
nums = [0,1,0,3,12]
```

过程：

| fast | nums[fast] | 操作 | slow | 数组 |
|---|---|---|---|---|
| 0 | 0 | 跳过 | 0 | [0,1,0,3,12] |
| 1 | 1 | 交换 | 1 | [1,0,0,3,12] |
| 2 | 0 | 跳过 | 1 | [1,0,0,3,12] |
| 3 | 3 | 交换 | 2 | [1,3,0,0,12] |
| 4 | 12 | 交换 | 3 | [1,3,12,0,0] |

最终：

```python
[1,3,12,0,0]
```

## 八、本题真正要学会的算法思想

# 双指针（Two Pointers）

核心：

```text
fast 找元素
slow 放元素
```

这是 LeetCode 最核心基础算法之一。

后面大量题目都会用：

- 删除元素
- 删除重复项
- 快慢指针
- 滑动窗口
- 原地数组修改

---

## 九、一句话记忆

```text
fast 负责遍历
slow 负责放非零元素
遇到非零就交换
```

---

## 一句话理解双指针

```text
fast 指针负责向前(从0向后面的元素)寻找非 0 元素，slow 指针负责把非 0 元素依次放到数组前面。
```

---

## 易懂例子

例如：

```python
nums = [0,5,0,2,8]
```

开始：

```python
slow = 0
```

---

### fast = 0

看到：

```python
0
```

是 0。

不交换。

数组：

```python
[0,5,0,2,8]
```

---

### fast = 1

看到：

```python
5
```

不是 0。

交换：

```python
nums[slow], nums[fast] = nums[fast], nums[slow]
```

```python
nums[0], nums[1]
```

数组变成：

```python
[5,0,0,2,8]
```

然后：

```python
slow += 1
```

现在：

```python
slow = 1
```

---

### fast = 3

看到：

```python
2
```

不是 0。

交换：

```python
nums[1], nums[3]
```

数组变成：

```python
[5,2,0,0,8]
```
slow += 1, -> nums[2]

---

### fast = 4

看到：

```python
8
```

不是 0。

交换：

```python
nums[2], nums[4]
```

数组变成：

```python
[5,2,8,0,0]
```

## 最终结果

```python
[5,2,8,0,0]
```

## 本质（真正核心）

```text
fast 负责找非 0，
```
