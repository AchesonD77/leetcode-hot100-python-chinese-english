# LeetCode 11. 盛最多水的容器（Container With Most Water）

![alt text](<../../Snap_for_Questions/2_TwoPointers_双指针/chinese_version/截屏2026-05-25 17.47.44.png>)

# 中文版本

## 一、题目核心

给定一个数组：

```python
height = [1,8,6,2,5,4,8,3,7]
```

每个数字表示一根柱子的高度。

选择两根柱子：

- 宽度 = 两根柱子的距离
- 高度 = 较短的柱子

求：

```text
最大的装水面积
```

---

## 二、面积计算公式（必须理解）

容器面积：

```text
面积 = 宽度 × 高度
```

这里：

```text
宽度 = right - left
高度 = min(height[left], height[right])
```

所以：

$$
Area = Width \times Height
$$

---

## 三、核心思想（双指针）

使用两个指针：

- `left`：左边开始
- `right`：右边开始

每次：

1. 计算当前面积
2. 更新最大值
3. 移动较短的柱子

---

## 四、为什么移动较短的柱子（重点）

**因为：容器高度由较短木板决定。**

```text
面积 = 宽度 × 较短高度
```

当前高度由：

```text
min(left_height, right_height)
```

决定。

如果移动较高柱子：

```text
高度不会变高
宽度还会变小
```

面积不可能更大。

所以：

```text
只能移动较短柱子
```

这是本题最核心思想。

---

## 五、Python 代码（必须背会）

```python
class Solution:
    def maxArea(self, height: list[int]) -> int:
        left = 0
        right = len(height) - 1

        max_area = 0

        while left < right:
            width = right - left

            current_height = min(height[left], height[right])

            area = width * current_height

            max_area = max(max_area, area)

            if height[left] < height[right]:
                left += 1
            else:
                right -= 1

        return max_area
```


返回较小值。


